# testing_nmap.py
You will need to first do:

````
sudo apt-get install nmap
sudo pip install python-nmap
````

You can then:
````
sudo python testing_nmap.py
````
Or run the current script:
````
sudo python whos_up.py
````
The code is fully commented.
